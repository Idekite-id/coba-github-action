== Local Machine Setup

. _Install Nodejs_ (_version >= 10_)
. _Install Yarn Package Manager_
. _Clone_ dari Git
. Arahkan _terminal_ ke _root directory project_ dan jalankan perintah `yarn install`
. _Run App_ menggunakan perintah `node .`
