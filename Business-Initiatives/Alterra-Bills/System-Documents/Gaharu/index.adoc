= Gaharu
:keywords: telco

== Overview

Gaharu merupakan sistem untuk _redeem_ voucher bagi partner Alterra. _Voucher_ yang di-_redeem_ dan ditukarkan dapat berupa voucher, paket data, token PLN dan E-Wallet (masih tahap _development_).
